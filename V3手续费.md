# V3 手续费

```
Pool 级别：
    feeGrowthGlobal0X128        → 全局 token0 手续费累积
    feeGrowthGlobal1X128        → 全局 token1 手续费累积

Tick 级别：
    feeGrowthOutside0X128       → tick 外侧 token0 累积
    feeGrowthOutside1X128       → tick 外侧 token1 累积

Position 级别：
    feeGrowthInside0LastX128    → 上次快照，区间内 token0
    feeGrowthInside1LastX128    → 上次快照，区间内 token1
    tokensOwed0                 → 待提取的 token0（手续费+本金）
    tokensOwed1                 → 待提取的 token1（手续费+本金）
```

当前token0 的 feeGrowthInside0 的计算  = global - below - above
